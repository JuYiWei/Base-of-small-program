3.1 视图容器

  • 总的来说，与 APP 开发的一些基本控件还是很像的；
  • 关于样式什么的都得靠自己写 WXSS，基本类似 CSS；
  • 组件提供一些额外的功能；
  • 从开发工具上来看，不建议直接写 style，而是写 WXSS 来控制样式；

一、view

  • 基础视图
属性 类型 默认值 说明 最低版本
hover-class String none 点击效果
hover-stop-propagation Boolean false 阻止父节点点击效果 1.5.0
hover-start-time Number 50 效果出发时间 ms
bover-stay-time Number 400 效果停留时间 ms 
.testView {
  background-color: red;
  color: white;
}

<view class='testView'>lalala</view>

二、srcoll-view

  • 滚动视图
属性 类型 默认值 说明
scroll-x Boolean false 横向滚动
scroll-y Boolean false 纵向滚动
upper-threshold Number 50 距离顶／左多远时（px），触发 scrolltoupper
lower-threshold Number 50 距离底／右多远时（px），触发 scrolltolower
scroll-top Number 纵向滚动条位置
scroll-left Number 横向滚动条位置
scroll-into-view String 滚动到某子元素 id
scroll-with-animation Boolean false 设置滚动条位置时使用动画
enable-back-to-top Boolean false iOS 点击顶部状态栏（安卓双击标题栏），滚回纵向顶部
bindscrolltoupper EventHandle 触发 scrolltoupper 事件
bindscrolltolower EventHandle 触发 scrolltolower 事件
bindscroll EventHandle 触发 scroll 事件
  • wxss
.testView {
  background-color: black;
  color: white;
}

.scrollViewRed {
  width: 100%;
  height: 400px;
  background-color: red;
}

.scrollViewGreen {
  width: 100%;
  height: 400px;
  background-color: green;
}
  • wxml
<view class='testView' bindtap="viewTap">lalala</view>

<scroll-view scroll-y style="height: 400px;" scroll-top="{{scrollTop}}" scroll-with-animation enable-back-to-top bindscrolltoupper="scrollTopHandle" bindscroll="scrollHandle">
  <view id="green" class="scrollViewRed"></view>
  <view id="red"  class="scrollViewGreen"></view>
</scroll-view>
  • js
data: {
    scrollTop: 0
  },

  viewTap: function(e) {
    console.log(this.data.scrollTop);
    this.setData({
      scrollTop: this.data.scrollTop == 400 ? this.data.scrollTop - 400 : this.data.scrollTop + 40
    });
  },
  scrollTopHandle: function(e) {
    console.log(e);
  },
  scrollHandle: function(e) {
    console.log(e);
  },

三、swiper

  • 带指示点的滚动视图（结合使用，变成横向滚动的广告横幅）
属性 类型 默认值 说明
indicator-dots Boolean false 显示指示点
indicator-color Color rgba(0,0,0,0.3) 指示点颜色
indicator-active-color Color #000000 选中时指示点颜色
autoplay Boolean false 自动切换
current Number 0 当前位置
interval Number 50000 自动切换时间间隔 ms
duration Number 500 滑动时长
circular Boolean false 是否衔接动画
vertical Boolean false 滑动方向
bindvhange EventHandle 触发 change 事件
  • wxss
.scrollViewRed {
  width: 100%;
  height: 400px;
  background-color: red;
}

.scrollViewGreen {
  width: 100%;
  height: 400px;
  background-color: green;
}
  • wxml
<swiper indicator-dots autoplay>
  <block wx:for="{{['scrollViewRed','scrollViewGreen','scrollViewRed','scrollViewGreen']}}">
    <swiper-item>
      <view class='{{item}}'></view>
    </swiper-item>
  </block>
</swiper>

四、movable-view

  • 可移动视图（应用场景较少）

  • wxml

<view class='testView' bindtap="viewTap">lalala</view>

<movable-area style="height: 300px;width: 300px;background: red;">
  <movable-view style="height: 50px; width: 50px; background: blue;" x="{{x}}" y="{{y}}" direction="all" inertia out-of-bounds>
  </movable-view>
</movable-area>
  • js
  data: {
    x: 0,
    y: 0
  },

  viewTap: function(e) {
    this.setData({
      x: this.data.x == 300 ? this.data.x - 300 : this.data.x + 30,
      y: this.data.y == 300 ? this.data.y - 300 : this.data.y + 30
    });
  }

五、cover-view 和 cover-image

  • 覆盖在原生组件之上的视图，包括 map、video、canvas，支持嵌套。
