4.6 媒体 - 视频

一、wx.chooseVideo(OBJECT)

  • 选择视频
参数 类型 必须 说明
sourceType StringArray 否 视频来源 相册 album，相机 camera，默认 ['album','camera']
maxDuration Number 否 拍摄时长，最长 60 s
camera String 否 前后摄像头选择 前 front，后 back，默认 back
success Function 否 接口成功回调
fail Function 否 接口失败回调
complete Function 否 接口完成回调
  • success
参数 说明
tempFilePath 临时文件路径
duration 视频长度 s
size 视频大小
height 长
width 宽
  • wxml
<view class="container">
    <video src="{{src}}"></video>
    <button bindtap="bindButtonTap">获取视频</button>
</view>
  • js
    bindButtonTap: function  (e) {
        var that = this
        wx.chooseVideo({
            sourceType: ['album', 'camera'],
            maxDuration: 60,
            camera: 'back',
            success: function (res) {
                that.setData({
                    src: res.tempFilePath
                })
            }
        })
    },

二、wx.saveVideoToPhotosAlbum(OBJECT)

  • 保存视频到相册
参数 类型 必须 说明
filePath String 是 图片路径，不支持网络图片
success Function 否 接口成功回调
fail Function 否 接口失败回调
complete Function 否 接口完成回调 
        wx.saveVideoToPhotosAlbum({
            filePath: res.tempFilePath,
            success: function (res) {
                console.log(res);
            },
            fail: function (err) {
                console.log(err);
            }
        });

三、视频组件控制

  • wx.createVideoContext(videoId)

  • 创建并返回 video 上下文 videoContext 对象，对应

  • 方法列表

方法 参数 说明 版本
play 播放
pause 暂停
seek position 跳转播放位置 s
sendDanmu danmuObj 发送弹幕 {text:'',color:''}
playbackRate rate 播放速率调节 1.4.0
requestFullScreen 进入全屏 1.4.0
exitFullScreen 退出全屏 1.4.0
  • wxml
<view class="section tc">
  <video id="myVideo" src="http://wxsnsdy.tc.qq.com/105/20210/snsdyvideodownload?filekey=30280201010421301f0201690402534804102ca905ce620b1241b726bc41dcff44e00204012882540400&bizid=1023&hy=SH&fileparam=302c020101042530230204136ffd93020457e3c4ff02024ef202031e8d7f02030f42400204045a320a0201000400"   enable-danmu danmu-btn controls></video>
  <view class="btn-area">
    <button bindtap="bindSendDanmu">发送弹幕</button>
  </view>
</view>
  • js
  onReady: function () {
        this.videoContext = wx.createVideoContext('myVideo')
  },
  
    bindSendDanmu: function  (e) {
        this.videoContext.sendDanmu({
            text: '今天天气好',
            color: 'red'
        })
    }
